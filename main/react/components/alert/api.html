Alert

Angular React Svelte
  • Examples
  • Api
  • Playground

Props

animation

Type: boolean
If `true`, alert closing will be animated.
Animation is triggered when clicked on the close button (×), via the `.close()` function or the visible prop is changed
Default value: true

animationOnInit

Type: boolean
If `true`, alert opening will be animated.
Animation is triggered when the `.open()` function is called or the visible prop is changed
Default value: false

ariaCloseButtonLabel

Type: string
Accessibility close button label
Default value: 'Close'

dismissible

Type: boolean
If `true`, alert can be dismissed by the user. The close button (×) will be displayed and you can be notified of the event with the (close) output.
Default value: true

onHidden

Type: () => void
Callback called when the alert is hidden.
Default value: 
noop

onShown

Type: () => void
Callback called when the alert is shown.
Default value: 
noop

onVisibleChange

Type: (visible: boolean) => void
Callback called when the alert visibility changed.
Default value: 
noop

slotDefault

Type: SlotContent<AlertContext>
Template for the alert content
Default value: undefined

slotStructure

Type: SlotContent<AlertContext>
Global template for the alert component
Default value: undefined

transition

Type: TransitionFn
The transition function will be executed when the alert is displayed or hidden.
Depending on the value of AlertProps.animationOnInit, the animation can be optionally skipped during the showing process.
Default value: fadeTransition

type

Type: string
Type of the alert. The are the following types: 'success', 'info', 'warning', 'danger', 'primary', 'secondary', 'light' and 'dark'.
Default value: 'primary'

visible

Type: boolean
If `true` the alert is visible to the user
Default value: true

Api

close

Type: () => void
Triggers alert closing programmatically (same as clicking on the close button (×)).

open

Type: () => void
Triggers the alert to be displayed for the user.
