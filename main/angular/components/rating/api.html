Rating

Props

ariaLabel

Type: string
The aria label
Default value: 'Rating'

ariaLabelledBy

Type: string
The aria labelled by
Default value: ''

ariaValueTextFn

Type: (rating: number, maxRating: number) => string
Return the value for the 'aria-valuetext' attribute.
Default value: 
(rating: number, maxRating: number) => `${rating} out of ${maxRating}`

className

Type: string
CSS classes to be applied on the widget main container
Default value: ''

disabled

Type: boolean
If `true`, the rating is disabled.
Default value: false

maxRating

Type: number
The maximum rating that can be given.
Default value: 10

onHover

Type: (rating: number) => void
An event emitted when the user is hovering over a given rating.
Event payload is equal to the rating being hovered over.
Default value: 
noop

onLeave

Type: (rating: number) => void
An event emitted when the user stops hovering over a given rating.
Event payload is equal to the rating of the last item being hovered over.
Default value: 
noop

onRatingChange

Type: (rating: number) => void
An event emitted when the rating is changed.
Event payload is equal to the newly selected rating.
Default value: 
noop

rating

Type: number
The current rating. Could be a decimal value like `3.75`.
Default value: 0

readonly

Type: boolean
If `true`, the rating can't be changed.
Default value: false

resettable

Type: boolean
Define if the rating can be reset.
If set to true, the user can 'unset' the rating value by cliking on the current rating value.
Default value: true

slotStar

Type: SlotContent<StarContext>
The template to override the way each star is displayed.
Default value: ({fill}) => String.fromCharCode(fill === 100 ? 9733 : 9734)

tabindex

Type: number
Allows setting a custom rating tabindex. If the component is disabled, `tabindex` will still be set to `-1`.
Default value: 0
