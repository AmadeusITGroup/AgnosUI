Props

animated

Type: boolean
If `true`, animates a striped progressbar. Takes effect only for browsers supporting CSS3 animations, and if `striped` is `true`.
Default value: false

ariaLabel

Type: string
The aria label.
Default value: 'Progressbar'

ariaValueTextFn

Type: (value: number, minimum: number, maximum: number) => string | undefined
Return the value for the 'aria-valuetext' attribute.
Default value: 
() => undefined

className

Type: string
CSS classes to be applied on the widget main container
Default value: ''

height

Type: string
Height of the progressbar, can be any valid css height value.
Default value: ''

max

Type: number
The maximum value.
Default value: 100

min

Type: number
The minimum value.
Default value: 0

slotContent

Type: SlotContent<ProgressbarContext>
Global template for the Progressbar content.
Default value: undefined

slotDefault

Type: SlotContent<ProgressbarContext>
Label of the progress.
Default value: undefined

striped

Type: boolean
If `true`, shows a striped progressbar.
Default value: false

value

Type: number
The current value.
Default value: 0
