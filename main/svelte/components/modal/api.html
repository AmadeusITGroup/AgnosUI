Props

animation

Type: boolean
Whether the modal and its backdrop (if present) should be animated when shown or hidden.
Default value: true

ariaCloseButtonLabel

Type: string
Value of the aria-label attribute to put on the close button.
Default value: 'Close'

backdrop

Type: boolean
Whether a backdrop should be created behind the modal.
Default value: true

backdropClass

Type: string
Classes to add on the backdrop DOM element.
Default value: ''

backdropTransition

Type: TransitionFn
The transition to use for the backdrop behind the modal (if present).
Default value: fadeTransition

className

Type: string
CSS classes to be applied on the widget main container
Default value: ''

closeButton

Type: boolean
Whether to display the close button.
Default value: true

closeOnOutsideClick

Type: boolean
Whether the modal should be closed when clicking on the viewport outside the modal.
Default value: true

container

Type: HTMLElement | null
Which element should contain the modal and backdrop DOM elements. If it is not null, the modal and backdrop DOM elements are moved to the specified container. Otherwise, they stay where the widget is located.
Default value: typeof window !== 'undefined' ? document.body : null

contentData

Type: Data
Data to use in content slots
Default value: undefined

modalTransition

Type: TransitionFn
The transition to use for the modal.
Default value: fadeTransition

onBeforeClose

Type: (event: ModalBeforeCloseEvent) => void
Event to be triggered when the modal is about to be closed (i.e. the close method was called).
Default value: 
noop

onHidden

Type: () => void
Event to be triggered when the transition is completed and the modal is not visible.
Default value: 
noop

onShown

Type: () => void
Event to be triggered when the transition is completed and the modal is visible.
Default value: 
noop

onVisibleChange

Type: (visible: boolean) => void
Event to be triggered when the visible property changes.
Default value: 
noop

slotDefault

Type: SlotContent<ModalContext<Data>>
Body of the modal.
Default value: undefined

slotFooter

Type: SlotContent<ModalContext<Data>>
Footer of the modal.
Default value: undefined

slotHeader

Type: SlotContent<ModalContext<Data>>
Header of the modal. The default header includes slotTitle.
Default value: undefined

slotStructure

Type: SlotContent<ModalContext<Data>>
Structure of the modal. The default structure uses slotHeader, slotDefault and slotFooter.
Default value: undefined

slotTitle

Type: SlotContent<ModalContext<Data>>
Title of the modal.
Default value: undefined

visible

Type: boolean
Whether the modal should be visible when the transition is completed.
Default value: false

Api

close

Type: (result?: any) => void
Closes the modal with the given result.

open

Type: () => Promise<any>
Opens the modal and returns a promise that is resolved when the modal is closed. The resolved value is the result passed to the close method and possibly changed by the onBeforeClose event handler

patch

Type: ModalWidget<Data>['patch']
Method to change some modal properties.
