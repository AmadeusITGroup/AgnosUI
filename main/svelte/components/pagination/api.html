Default value: 'Action link for next page'

ariaPageLabel

Type: (processPage: number, pageCount: number) => string
Provide the label for each "Page" page button. This is used for accessibility purposes. for I18n, we suggest to use the global configuration override any configuration parameters provided for this
Default value: 
(processPage: number, pageCount: number) => `Page ${processPage} of ${pageCount}`

ariaPreviousLabel

Type: string
The label for the "Previous" page button. for I18n, we suggest to use the global configuration override any configuration parameters provided for this
Default value: 'Action link for previous page'

boundaryLinks

Type: boolean
If `true`, the "First" and "Last" page links are shown.
Default value: false

className

Type: string
An input to add a custom class to the UL
Default value: ''

collectionSize

Type: number
The number of items in your paginated collection.
Note, that this is not the number of pages. Page numbers are calculated dynamically based on `collectionSize` and `pageSize`.
Ex. if you have 100 items in your collection and displaying 20 items per page, you'll end up with 5 pages.
Whatever the collectionSize the page number is of minimum 1.
Default value: 0

directionLinks

Type: boolean
If `true`, the "Next" and "Previous" page links are shown.
Default value: true

disabled

Type: boolean
If `true`, pagination links will be disabled.
Default value: false

onPageChange

Type: (page: number) => void
An event fired when the page is changed.
Event payload is the number of the newly selected page.
Page numbers start with `1`.
Default value: 
noop

page

Type: number
The current page.
Page numbers start with `1`.
Default value: 1

pageSize

Type: number
The number of items per page.
Default value: 10

pagesFactory

Type: (page: number, pageCount: number) => number[]
pagesFactory returns a function computing the array of pages to be displayed as number (-1 are treated as ellipsis). Use Page slot to customize the pages view and not this
Default value: 
(page: number, pageCount: number) => {
		// TODO extract this for testing
		const pages: number[] = [];
		for (let i = 1; i <= pageCount; i++) {
			pages.push(i);
		}
		return pages;
	}

size

Type: 'sm' | 'lg' | null
The pagination display size.
Bootstrap currently supports small and large sizes.
Default value: null
