Represents a server-side rendered HTML element with limited functionality.

This interface extends a subset of the [HTMLElement](https://developer.mozilla.org/docs/Web/API/HTMLElement) interface, providing
methods to set and remove attributes, manipulate the element's classes, and
partially manipulate the element's style.

It inherits the [setAttribute](https://developer.mozilla.org/docs/Web/API/Element/setAttribute) and [getAttribute](https://developer.mozilla.org/docs/Web/API/Element/getAttribute) methods from the [HTMLElement](https://developer.mozilla.org/docs/Web/API/HTMLElement) interface.
