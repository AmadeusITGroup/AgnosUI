Progressbar

  • Examples
  • Api
  • Playground

Props

animated

Type:
boolean
Default value:
false
If `true`, animates a striped progressbar. Takes effect only for browsers supporting CSS3 animations, and if `striped` is `true`.

ariaLabel

Type:
string
Default value:
'Progressbar'
The aria label.

ariaValueTextFn

Type:
(value: number, minimum: number, maximum: number) => string
Default value:
() => undefined
Return the value for the 'aria-valuetext' attribute.

className

Type:
string
Default value:
''
CSS classes to be applied on the widget main container

height

Type:
string
Default value:
''
Height of the progressbar, can be any valid css height value.

max

Type:
number
Default value:
100
The maximum value.

min

Type:
number
Default value:
0
The minimum value.

slotDefault

Type:
SlotContent<ProgressbarContext>
Default value:
undefined
Label of the progress.

slotStructure

Type:
SlotContent<ProgressbarContext>
Default value:
undefined
Global template for the Progressbar.

striped

Type:
boolean
Default value:
false
If `true`, shows a striped progressbar.

type

Type:
"primary" | "success" | "info" | "warning" | "danger" | "secondary" | "light" | "dark"
Default value:
undefined
Type of the progressbar, following bootstrap types.

value

Type:
number
Default value:
0
The current value.
